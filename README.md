# Turneringshåndterings-GUI

## Værktøj

- Node.js - 22 LTS
- .NET - 10.0

## For at køre projektet

```bash
cd ./guiprojekt.client
npm install
cd ..
cd ./GUIProjekt.Server
dotnet run --launch-profile https
```
